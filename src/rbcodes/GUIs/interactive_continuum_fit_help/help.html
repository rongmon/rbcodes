Interactive Continuum Fitting — Help

Fit a continuum to spectral data by masking absorption/emission features and applying either polynomial or spline fitting. The normalized spectrum is shown in the bottom panel after fitting.
Important: ensure no matplotlib toolbar button (zoom/pan) is active when clicking to add masks or spline points.

Basic Workflow

  1. Choose Polynomial or Spline fitting method from the sidebar.
  2. Zoom/pan to the region of interest using keyboard shortcuts or the matplotlib toolbar.
  3. Polynomial: left-click twice to add masks over absorption/emission features, then click Fit Continuum.
  4. Spline: left-click to place median anchor points (or press b for exact placement), then click Fit Continuum.
  5. Inspect the normalized spectrum in the bottom panel. Adjust masks or anchor points and refit.
  6. Click Accept & Return (or press a) when satisfied.

Fitting Methods

Polynomial Fitting

Fits a Legendre polynomial to the unmasked regions. Best for spectra with smooth, slowly-varying continua.

OptionDescription
Polynomial OrderFixed order to use when Auto Optimize is off. Higher orders fit more complex shapes but risk overfitting.
Use WeightsWeight the fit by 1/error².
Auto OptimizeUses BIC to find the best order within the Min/Max Order range (set in the Advanced tab).
Start with Auto Optimize enabled and a range of 1–6. Increase Max Order only for genuinely complex continua.

Spline Fitting

Cubic spline interpolation through user-placed anchor points. Better for irregular continua or when polynomial fitting struggles.

OptionDescription
Spline DegreeOrder of polynomial pieces (3 = cubic, recommended).
Smoothing0 = exact interpolation through all points; larger values allow the curve to deviate for smoothness.
Median WindowSize of window used to compute the median flux when placing a point with a left-click.
Window TypeWhether the median window size is in pixels or axis units (km/s or Å).

Adding / removing spline anchor points

  • Left-click on the top panel: places a point at the median flux within the window around the cursor.
  • b: places a point at the exact cursor position.
  • Right-click: removes the closest anchor point.
  • c: clears all anchor points.
At least 3 anchor points are required. Place them on true continuum regions, not on features.

Working with Masks

Masks exclude regions from the polynomial fit (they are not used in spline fitting).

ActionHow
Add maskLeft-click twice to define the start and end of the region.
Remove mask(s)Right-click twice; any mask overlapping the selected range is removed.
Undo last maskClick Undo Last or press z.
Reset all masksClick Reset Masks or press r.
Manual entryClick Manual Entry or press m to type exact boundaries.
Auto-maskClick Auto-Mask: detects features via MAD sigma-clipping and error thresholding.
Overlapping masks are automatically merged into a single contiguous region.

Auto-Mask parameters (Advanced tab)

  • Auto-Mask Sigma: detection threshold in units of the robust (MAD) standard deviation.
  • Min Mask Width: minimum feature width to be included.
  • Grouping Window: nearby regions within this distance are merged into a single mask.

Tips & Best Practices

  • Prefer cursor-based zoom (x/X, t/b) over symmetric zoom — it lets you home in on a specific feature without repositioning repeatedly.
  • Polynomial: start with Auto Optimize (order range 1–6). Increase Max Order only for genuinely complex continua.
  • Spline: place anchor points at regular intervals on true continuum; avoid features. Use the Median Window to reduce noise sensitivity.
  • Masking: be generous with mask boundaries — better to exclude a bit of clean continuum than to include part of a feature in the fit.
  • Inspect the bottom panel: after fitting, the normalized spectrum should hover near 1.0 in continuum regions. Systematic dips or bumps indicate the fit needs adjustment.
  • Use weights (polynomial mode) when the noise level varies significantly across the spectrum.
Common pitfalls:
  • Polynomial order too high → wiggles between masked regions (overfitting).
  • Spline anchor points placed on features → the continuum is pulled toward the feature.
  • Masks too narrow → edges of absorption features bias the polynomial fit.
  • Forgetting to deactivate the matplotlib zoom/pan tool before clicking to add masks.
Remember: continuum accuracy directly affects equivalent width measurements and column density calculations. A few extra seconds refining the fit is always worthwhile.
